## 📃 Init truffle

Step 1: Setup Smart contract (Truffle and Ganache)
```sh
  npm i -g truffle
  install Ganache
  open Ganache
  create new Workspace
  import truffle-config.js
  Start

  truffle compile
  truffle migrate --reset

```
Step 2: Start Client
```sh
  cd client
  npm install
  npm run dev
```
